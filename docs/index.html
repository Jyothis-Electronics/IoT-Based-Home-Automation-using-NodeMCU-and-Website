About This Project

This IoT-based Home Automation System is designed and developed by Jyothish Electronics. The project leverages the power of ESP8266 NodeMCU microcontroller to provide seamless control of home appliances through a modern web interface.

The system features a relay-based control mechanism that allows you to switch electrical devices on and off remotely. Whether you're at home or away, you can monitor and control your devices through any web browser on your smartphone, tablet, or computer.

Key Highlights:

  • WiFi-enabled remote control system
  • Real-time device status monitoring
  • Responsive web interface
  • Solar monitoring capability
  • Easy to install and configure
  • Expandable for multiple devices
